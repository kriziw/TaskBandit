# TaskBandit Brand Guidelines

## Personality

- playful
- upbeat
- warm
- lightly mischievous
- clear and encouraging

## Mascot

TaskBandit uses a cute raccoon mascot with a friendly bandit look. The character should feel helpful and energetic rather than sneaky or chaotic.

## Core Palette

- Ink: `#243043`
- Cream: `#FFF7E8`
- Sky: `#73C7F4`
- Mint: `#73D9B3`
- Gold: `#FFC94A`
- Coral: `#FF7A6B`

## Voice

- "Nice work, bandit."
- "One more task and you are done for today."
- "This one needs a quick fix before approval."

## UI Principles

- bright, roomy layouts
- playful accents without clutter
- gamification should feel rewarding, not noisy
- admin tools should stay readable and calm even within the colorful brand
